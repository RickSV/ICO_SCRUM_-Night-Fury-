Conceptos basicos de la ingeniera de software

Ingenieraa de software

Es la aplicacion de enfoque sistematico, disciplinado y cuantificable hacia el desarrollo, operacion y mantenimiento del software.

Fase de desarrollo de software

Conjunto de principios basicos que gobiernan cada area de la tecnologaa e incluyen actividades de modelado y otras tecnicas descriptivas.
Fases de desarrollo del software:
-Analisis y definicion de requerimientos.
-Diseño del sistema.
-Desarrollo.
-Pruebas del sistema

Analisis y definicion de requerimientos.

Se refiere a entender el problema y capturar los requerimientos funcionales, entidades y actividades.

Diseño de sistema.

Proceso creatico donde se determina las tecnologias de programacion y computo y se define la estructura del producto.

Calidad de software.

Es aquel que cumple con los requerimientos del usuario final, es decir el software entregado funciona como se espera que funcione.

Herramienta en la Ingenieria de software.

Instrumento o sistema automatizado para realizar de manera más exacta, eficiente y productiva una tarea durante el ciclo de vida de desarrollo de software.

Proceso en la ingenieria de software.

Es la definicion de un conjunto de actividades formales que permiten transformar los requisitos de un cliente/usuario en un sistema software.
